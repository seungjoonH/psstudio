# Changelog


## v1.0.6

v1.0.6 릴리즈 노트입니다.

<br />

### 과제를 누가 등록했는지 확인하고 골라 볼 수 있습니다.

- 과제 목록, 상세, 캘린더에서 **과제 등록자**를 확인합니다.
- 필터에서 등록자를 골라 내가 보고 싶은 과제만 모아 볼 수 있습니다.

<br />

### 과제 목록과 캘린더에서 진행 상황을 더 쉽게 읽습니다.

- 과제 카드에서 제목·플랫폼·난이도 칩과 D-day·해결 여부를 양쪽으로 나눠 보여 줍니다.
- 그룹 제출 진행도(제출 n/m)와 진행 막대를 카드 오른쪽 아래에서 확인합니다.
- 캘린더에서 날짜를 눌렀을 때 뜨는 과제 목록도 같은 정보 구조로 정리됩니다.
- 홈 화면 할 일 칸반 카드 레이아웃도 과제 목록과 맞춰 두었습니다.

<br />

### 제출한 코드를 더 빠르게 복사할 수 있습니다.

- 제출 상세의 코드 영역 오른쪽 위에서 **코드 복사** 버튼을 누르면 전체 코드가 클립보드에 복사됩니다.

<br />

### Google 로그인 실패 시 안내가 더 분명해집니다.

- Google OAuth 처리 중 문제가 생기면, 가능한 경우 Google이 돌려준 오류 설명을 함께 보여 줍니다.
- 로그인이 막혔을 때 원인을 추측하기 어렵던 경우가 줄어듭니다.

<br />

## v1.0.5

v1.0.5 릴리즈 노트입니다.

<br />

### AI 자동 채우기가 LeetCode 문제를 더 안정적으로 읽습니다.

- LeetCode 문제 링크를 넣었을 때, 이제 문제 제목과 난이도, 본문을 더 안정적으로 가져옵니다.
- `/description/`이 붙은 링크나 daily question처럼 쿼리 값이 붙은 링크도 더 자연스럽게 처리됩니다.
- LeetCode 문제에서 자동 채우기가 빈 내용으로 실패하던 경우가 줄어듭니다.
- 그래서 LeetCode 문제를 과제로 만들 때 제목, 힌트, 난이도를 한 번에 채우는 흐름이 더 안정적으로 동작합니다.

<br />

### 문제 링크 인식이 더 넓어집니다.

- Programmers School 링크도 문제 링크로 더 정확하게 인식합니다.
- LeetCode 링크는 불필요한 경로나 쿼리를 정리해서 같은 문제를 더 일관되게 다룹니다.

<br />

## v1.0.4

v1.0.4 릴리즈 노트입니다.

<br />

### 홈 화면에서 최근 제출을 더 알아보기 쉽습니다.

- 홈 화면의 최근 제출 영역에서 제출 제목뿐 아니라 어떤 과제에 낸 제출인지 더 분명하게 확인합니다.
- 그래서 제출 제목이 비슷하거나 짧아도, 어떤 과제에 대한 제출인지 더 쉽게 구분하게 됩니다.

### 제출 상세에서 댓글 작성 흐름이 더 자연스러워집니다.

- 제출 상세에서 댓글을 등록하면 작성창이 정리되고 화면이 바로 갱신됩니다.
- 댓글을 남긴 뒤에도 입력하던 내용이 그대로 남아 보이던 흐름이 줄어듭니다.

## v1.0.3

v1.0.3 릴리즈 노트입니다.

<br />

### 과제 제목과 알림이 화면 안에서 더 잘 읽힙니다.

- 긴 과제 제목을 만들 때 화면에서 다루기 쉬운 길이로 제한됩니다.
- 새 과제 알림이나 마감 임박 알림에서 긴 과제명이 화면을 과하게 밀어내는 문제가 줄어듭니다.
- 알림 문구가 한국어와 영어 화면에 맞게 더 자연스럽게 표시됩니다.

<br />

### 과제 생성과 설정 화면이 더 쓰기 쉬워집니다.

- 과제를 만들거나 수정할 때 알고리즘 태그를 더 편하게 입력합니다.
- 과제 설정, 삭제 확인, 마감 정보 같은 관리 화면의 흐름이 더 정리됩니다.
- 과제 목록과 진행 화면에서도 문제 플랫폼, 난이도, 알고리즘 정보를 더 일관되게 확인합니다.

<br />

### 캘린더와 홈 화면에서 과제 상태를 더 쉽게 파악합니다.

- 캘린더에서 과제 제목과 상태를 더 안정적으로 확인합니다.
- 홈 화면의 할 일 영역에서 그룹명이나 과제명이 길어도 내용을 더 쉽게 구분하게 됩니다.
- 그룹 안에서 이동할 때 현재 위치와 과제 관련 메뉴를 더 자연스럽게 확인합니다.

<br />

### AI 피드백 요청 권한이 더 명확해집니다.

- 이제 제출 코드에 대한 AI 피드백은 해당 제출을 작성한 사람이 요청하는 흐름으로 정리됩니다.
- 그래서 다른 사람의 제출에 대해 의도치 않게 AI 피드백을 요청하는 상황이 줄어듭니다.

<br />

## v1.0.2

v1.0.2 릴리즈 노트입니다.

<br />

### 마감 시간과 일정 선택이 더 정확해집니다.

- 과제의 마감 일시를 저장하고 다시 불러올 때, 이제 한국 시간 기준으로 더 안정적으로 맞춰집니다.
- 그래서 `23:59`처럼 설정한 시간이 새로고침 뒤 엉뚱한 시각으로 바뀌어 보이던 문제가 줄어듭니다.
- 과제 수행 기간이나 마감 요일을 바꾸면 마감 일시가 함께 바뀌고, 반대로 마감 일시를 직접 바꾸면 기간/요일 선택도 같이 맞춰집니다.
- 이미 지난 시각을 고르면 마감 요일이 계속 선택된 것처럼 보이던 흐름도 더 자연스럽게 정리됩니다.

## v1.0.1

v1.0.1 릴리즈 노트입니다.

<br />

### 내가 해야 할 과제가 더 분명해집니다.

- 이제 과제를 만들 때 그룹 전체가 아니라 특정 사람만 대상으로 정할 수 있습니다.
- 그래서 홈 화면, 그룹 화면, 진행률 화면에서 `내가 해야 하는 과제`가 더 정확하게 보입니다.
- 예전보다 다른 사람 과제와 내 과제가 덜 섞여 보여서, 할 일을 찾기 쉬워집니다.

<br />

### 진행률과 캘린더가 더 정확하고 이해하기 쉬워집니다.

- 그룹 진행률은 이제 `실제로 이 과제를 해야 하는 사람들`을 기준으로 계산합니다.
- 그래서 진행률 숫자가 더 현실적으로 보이고, 누가 아직 안 했는지도 더 자연스럽게 이해할 수 있습니다.
- 캘린더에서도 과제 상태가 더 또렷하게 보입니다. 해결한 과제, 아직 안 한 과제, 늦게 낸 과제가 전보다 더 잘 구분됩니다.
- 캘린더에서 과제를 눌렀을 때도 출처, 난이도, 마감 정보를 더 쉽게 확인할 수 있습니다.

<br />

### 알림이 더 빠르고 찾기 쉬워집니다.

- 새 알림이 생기면 화면에 더 빠르게 반영됩니다.
- 앱 안에서 알림 페이지로 들어가는 길도 더 잘 보입니다.
- 과제 마감이 가까워졌을 때 알려주는 흐름도 더 자연스러워져, 중요한 마감을 놓칠 가능성이 줄어듭니다.

<br />

### AI 자동 채우기가 더 자연스러워집니다.

- 과제 링크를 넣으면 제목, 힌트, 난이도 같은 정보를 더 자연스럽게 자동으로 채울 수 있습니다.
- 이제 LeetCode 같은 문제 사이트에서도 AI 자동 채우기가 지원됩니다.
- 한국어로 쓰는 사람은 한국어 힌트를, 영어로 쓰는 사람은 영어 힌트를 받을 수 있습니다.

<br />

### 화면이 전체적으로 더 깔끔해집니다.

- 아이콘 모양이 통일되어 화면이 더 정돈되어 보입니다.
- 버튼이나 카드에 마우스를 올렸을 때의 움직임도 더 일관되게 보입니다.
- 팝업창이 뜰 때 보이던 어색한 스크롤바 문제가 줄어듭니다.
- 없는 페이지로 들어갔을 때 보이는 안내 화면도 더 알아보기 쉬워집니다.
