미제출 과제 목록, 최근 제출한 풀이, 활동 알림을 한 화면에서 파악합니다.

### 3. 그룹 기반 스터디 운영

여러 그룹에 동시에 참여하고 그룹별 멤버 수와 남은 과제 수를 한 페이지에서 파악합니다. <br />
초대 코드와 초대 링크로 스터디원을 모으고, **OWNER** / **MANAGER** / **MEMBER** 역할에 따라 그룹 설정과 과제 생성 권한을 나눕니다.

### 6. diff 기반 코드 리뷰

제출 코드는 버전별로 보관되고, diff 화면에서 변경된 라인을 바로 확인합니다. <br />
라인 범위를 선택해 리뷰를 남기고, **AI 피드백 받기** 버튼으로 제출 코드 기반 AI 리뷰를 즉시 요청합니다.

<img width="777" alt="diff 기반 코드 리뷰" src="https://github.com/user-attachments/assets/471f2a8b-5e45-4833-9da9-4c534b212969" />

<br clear="both" />

### 7. 동료 리뷰 + AI 튜터 리뷰

동료의 코드 리뷰와 AI 튜터 리뷰가 같은 diff 화면 안에 함께 표시됩니다. <br />
AI는 라인 범위와 코드 문맥을 분석해 개선 포인트를 구체적인 코멘트로 남깁니다. <br />
제출 노트는 Markdown, 수식, 코드 하이라이트를 지원합니다.

**GROUP** &nbsp; **ASSIGN** &nbsp; **REVIEW** &nbsp; **ANALYZE**

<p>
<sup><strong>GROUP</strong> - OAuth 로그인 후 그룹을 만들고 코드, 링크, 이메일 초대로 멤버를 모읍니다.</sup><br />
<sup><strong>ASSIGN</strong> - 문제 URL, 마감, 대상자, 난이도, 알고리즘 태그를 과제로 관리합니다.</sup><br />
<sup><strong>REVIEW</strong> - 제출 버전과 diff를 기준으로 사람의 라인 리뷰와 댓글을 쌓습니다.</sup><br />
<sup><strong>ANALYZE</strong> - AI 리뷰와 코호트 분석으로 여러 풀이의 차이를 요약합니다.</sup>

### 설치 및 실행

```sh
# 저장소 클론
git clone https://github.com/seungjoonH/psstudio.git

# 프로젝트 루트로 이동
cd psstudio

# 의존성 설치
pnpm install

# 환경 변수 파일 생성
cp be/env.sample be/.env.local
cp fe/env.smaple fe/.env.local
cp be/env.sample worker/.env.local

# 로컬 DB와 Redis 실행
docker compose up -d supabase redis

# 마이그레이션과 시드 실행
pnpm db:migrate
pnpm db:seed

# 개발 서버 실행
pnpm dev
```

<br />

### 기본 포트

- Frontend: `http://localhost:3100`
- Backend: `http://localhost:4000`
- PostgreSQL: `54322`
- Redis: `6379`

<br />

### 환경변수

```env
# ─── Backend
BE_PORT=4000
BE_PUBLIC_BASE_URL=http://localhost:4000
FE_PUBLIC_BASE_URL=http://localhost:3100

SESSION_COOKIE_NAME=psss
SESSION_SECRET=

DATABASE_URL=postgres://postgres:postgres@localhost:54322/postgres
REDIS_URL=redis://localhost:6379

GOOGLE_OAUTH_CLIENT_ID=
GOOGLE_OAUTH_CLIENT_SECRET=
GOOGLE_OAUTH_REDIRECT_URI=http://localhost:4000/api/v1/auth/oauth/google/callback

GITHUB_OAUTH_CLIENT_ID=
GITHUB_OAUTH_CLIENT_SECRET=
GITHUB_OAUTH_REDIRECT_URI=http://localhost:4000/api/v1/auth/oauth/github/callback

LLM_PROVIDER=openrouter
OPEN_ROUTER_API_KEY=

REQUESTY_BASE_URL=https://router.requesty.ai/v1/chat/completions
LLM_MODEL_PROBLEM_ANALYZE=openai/gpt-5-mini
LLM_MODEL_ASSIGNMENT_AUTOFILL=openai/gpt-4o-mini
LLM_MODEL_SUBMISSION_REVIEW=openai/gpt-5-mini

RESEND_API_KEY=
EMAIL_FROM_ADDRESS=
AI_TOKEN_DEFAULT=100000

# ─── Frontend
NEXT_PUBLIC_API_BASE_URL=http://localhost:4000
```
