# FE Phase 2 작업 트랙

이 문서는 Phase 2 인증/사용자 구현의 FE 진행 기준입니다.

## 목표

- 로그인 시작 화면에서 Google/GitHub OAuth 진입 제공.
- 로그인 후 내 프로필(닉네임/이미지) 조회와 닉네임 수정 UI 제공.
- 탈퇴 액션 UI와 결과 처리 제공.

## 고정 정책

- 로그인 수단은 OAuth만 제공합니다.
- provider별 계정은 별도 사용자로 표시합니다.
- 프로필 이미지는 로그인 시마다 최신값으로 갱신됩니다.
- 닉네임은 중복 허용입니다.
